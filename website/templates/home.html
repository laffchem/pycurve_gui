Instructions

Choose the csv file you wish to use from your computer.

Open the csv and verify the following information or the program will not operate correctly.

  • The column name that is to be curved. (SUPER IMPORTANT!)
  • The grades are in values of percent. If they are points, this won't curve correctly.
  • The file is in csv format. If it is not, the program won't allow you to upload it.
  • All fields are filled out before hitting submit. If not, you will have to refill the form and upload the file.

If you are unsure of how to make your file a csv, click the link below.

How to create a csv!

These are the curve options.

  • Option 1 -> The square root curve will take the square root of each score and multiply it by 10.
  • Option 2 -> The max score curve will see which grade range the top score falls in between and subtract the max score from the upper number. This modifier will be added to all grades.
  • Option 3 -> This will just add 10 points to all the scores
